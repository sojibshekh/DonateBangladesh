How do you select an element from the DOM?

4 way to select an element from the DOM

  • 1. getElementById
  • 2. getElementsByClassName
  • 3. getElementsByTagName
  • 4. querySelector and querySelectorAll

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. This is efficient, simplifies handling dynamic content, and keeps the code cleaner.
